Learning Objectives [# STAT 301 3 Credits, Probability & Statistics ](https://pawar1550.wixsite.com/claflin-courses/biol341).
Probability & Statistics I & II, Biostatistics, Data Analysis (topics including descriptive & inferential statistics, variable types (qualitative, quantitative), distribution properties, discrete & continuous types, R functions, probability distributions, Hypothesis testing using critical value (Probability, F Statistics, Coefficient) using ANOVA, T-test, correlation, chi-Square test, regression, etc.). Regression (linear, multiple, logistic), Probability distributions (Discrete, Binomial, Poisson, Continuous Uniform, Chi-squared, Student-t, Fisher,)  Exponential, Normal, Chi-Square), Hypothesis testing using critical value (Probability, F Statistics, Coefficient) using Analysis of Variance (ANOVA), data wrangling, Post Hoc Test. Data wrangling and handling. Data Visualization (BoxPlots, Scatter Plots).
